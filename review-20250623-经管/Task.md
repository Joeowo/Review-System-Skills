# 学习任务

## Task 1: 项目管理基础 ✅

**状态**: completed
**来源**: sources/12-项目管理.md 第1-7页
**Q&A轮次**: 1轮（15个问题）
**学前问卷**:
- 什么是项目？项目与运作有何区别？
- 项目干系人包括哪些？
- 项目生命周期一般划分哪几个阶段？
**备注**: 涵盖项目管理基本概念、项目与运作的区别、项目生命周期、关键成功因素

**学习要点**:
- **项目定义**: 为获得独特产品/服务/成果的临时性工作，具有临时性和独特性
- **项目特征**: 多个干系人、工作相互依赖、资源冲突、不确定性（4个典型特征）
- **产出类型**: 产品/服务/成果（可能失败）+ 项目过程资产（必然产生）
- **项目vs运作**: 持续性（临时vs持续）、目标（达成vs维持）、预算（起止期vs周期）、不确定性（大vs小）
- **项目管理**: 计划、组织、领导、控制四大手段，确保项目成功
- **制约目标**: 时间、成本、质量（铁三角）
- **生命周期**: 启动（项目章程）→规划→实施（风险最大）→收尾（验收总结）
- **CSF**: 高层领导支持、清晰目标、良好计划、有效沟通、高效团队

## Task 2: 项目管理技术与工具 ✅

**状态**: completed
**来源**: sources/12-项目管理.md 第8-18页
**Q&A轮次**: 1轮（14个问题）
**学前问卷**:
1. 什么是工作分解结构（WBS）？它是以什么为导向的工作层级分解？
2. 什么是挣值分析（EVA）？它综合哪三个要素客观测量项目进展？
3. RACI矩阵中四个字母分别代表什么？一项工作应该仅有一个什么人？
4. 项目活动之间的时间依赖关系有哪四种类型？默认是哪种关系？
5. CPM中如何计算活动的最早开始日期(ES)和最晚完成日期(LF)？
6. CPM关键路径法的完整计算步骤是什么？
7. 在挣值分析中，如何判断项目成本和进度的绩效？
**备注**: 涵盖WBS、RAM、CPM、EVA等核心项目管理技术

**学习要点**:
- **WBS**: 以可交付物为导向的工作层级分解，最低层次为工作包
- **WBS五原则**: 包括全部工作、分解程度适度、工作包明确要素、唯一编码、附加工作说明书
- **RACI**: R(执行)、A(负责)、C(咨询)、I(通知)，A一般仅一个
- **时间关系**: FS(完成到开始)、SS(开始到开始)、FF(完成到完成)、SF(开始到完成)，默认FS0
- **CPM公式**: ES=max{EF_前+FS}、EF=ES+DU、LF=min{LS_后-FS}、LS=LF-DU
- **时差**: TF=LS-ES=LF-EF（总时差）、FF=min{ES_后-FS-EF}（自由时差）
- **关键路径**: TF=0的活动构成的路径，项目最长路径，决定工期
- **EVA**: PV(计划价值)、EV(挣值)、AC(实际成本)
- **EVA指标**: CV=EV-AC、CPI=EV/AC、SV=EV-PV、SPI=EV/PV
- **EAC三种方式**: BAC/CPI(按趋势)、实际+重估(原失效)、实际+剩余预算(特殊情况)

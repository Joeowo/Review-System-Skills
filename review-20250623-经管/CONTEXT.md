## Relationships

### 项目管理与经济评价的关系
- 项目管理关注如何在有限资源下高效完成项目
- 投资项目经济评价关注项目在经济上是否可行
- 两者相辅相成：项目管理需要经济评价作为决策依据，经济评价的结果需要通过项目管理来实现

### 决策分析与其他两者的关系
- 决策分析是一种方法论，可用于项目管理中的各种决策
- 项目经济评价本质上是一种投资决策问题，需要应用决策分析方法

## 术语定义

### 项目管理术语
| 术语 | 定义 |
|------|------|
| 项目 | 为获得独特产品、服务或成果而实施的临时性工作，具有临时性和独特性 |
| 运作 | 组织中持续重复的工作，内容规范、流程化、制度化 |
| 干系人 | 影响项目或受项目影响的人 |
| 项目生命周期 | 项目从开始到结束的过程，一般分为启动、规划、实施、收尾四个阶段 |
| WBS | 工作分解结构（Work Break-down Structure），以可交付物为导向的工作层级分解 |
| RAM | 责任分配矩阵（Responsibility Assignment Matrix），将WBS与工作干系人联系起来的二维表 |
| RACI | RAM的表示形式：R(执行)、A(负责)、C(咨询)、I(通知) |
| CPM | 关键路径法（Critical Path Method），用于确定项目关键路径和工期的技术 |
| 关键路径 | 总时差为零的活动构成的路径，直接决定项目工期 |
| 挣值分析 | 将范围、进度和资源综合起来，客观测量项目进展的绩效管理方法 |
| PV | 计划价值（Planned Value），为计划工作分配的预算 |
| EV | 挣值（Earned Value），已完成工作以分配预算为标准折算的价值 |
| AC | 实际成本（Actual Cost），已完成工作消耗的实际费用 |

### 投资项目经济评价术语
| 术语 | 定义 |
|------|------|
| 固定资产 | 使用期限较长、单位价值较高、能在使用过程中保持原有物质形态的资产 |
| 无形资产 | 不具有实物形态、能为企业长期使用并提供权利或利益的资产 |
| 递延资产 | 集中发生的、需要在以后年度内分期摊销的费用 |
| 流动资产 | 生产经营期内投入的维持项目运营的周转资金 |
| 折旧 | 固定资产由于使用磨损或时间陈旧等因素造成价值降低 |
| 净现金流 | 某期间流入和流出项目的资金流之差 |
| 资金时间价值 | 资金在运动过程中的增值或不同时间点上等额资金在实际价值上的差别 |
| 现值(P) | 发生或折算在时间序列起始时刻的资金价值 |
| 终值(F) | 发生或折算在时间序列终点时刻的资金价值 |
| 年金(A) | 在时间序列内，每隔相同时间都会发生的资金价值 |
| 静态投资回收期 | 不考虑资金时间价值，以项目净收益抵偿全部投资所需时间 |
| 动态投资回收期 | 考虑资金时间价值，按基准收益率收回投资所需时间 |
| 净现值(NPV) | 按一定折现率，将项目寿命期内各年净现金流量折现到基准年的现值之和 |
| 内部收益率(IRR) | 净现值为零时的折现率 |

### 决策分析术语
| 术语 | 定义 |
|------|------|
| 决策者 | 决策主体，可以是个体或群体 |
| 行动方案 | 实现决策目标所采取的具体措施和手段 |
| 自然状态 | 采取某种决策方案时客观存在的各种环境状态 |
| 条件结果 | 采取某种行动方案在不同自然状态下所出现的结果 |
| 确定型决策 | 自然状态和条件结果完全被决策者掌握的决策 |
| 不确定型决策 | 自然状态和条件结果具有偶然性但无法度量的决策 |
| 风险型决策 | 自然状态和条件结果具有偶然性但发生概率可测的决策 |
| 乐观准则 | 好中取好的决策态度 |
| 悲观准则 | 从最坏情况中选择最好的决策态度 |
| 等概率准则 | 假设各自然状态发生概率相等的决策方法 |
| 先验概率 | 基于历史数据或经验估计的无条件概率 |
| 后验概率 | 通过新信息修正先验概率得到的条件概率 |
| 贝叶斯风险 | 策略的期望损失 |
| EVSI | 采样信息的期望价值 |
| EVPI | 完全信息的期望价值 |

## Task 2: 学前问卷

**主题**: 项目管理技术与工具

**核心问题**:
1. 什么是工作分解结构（WBS）？它是以什么为导向的工作层级分解？
2. 什么是挣值分析（EVA）？它综合哪三个要素客观测量项目进展？
3. RACI矩阵中四个字母分别代表什么？一项工作应该仅有一个什么人？
4. 项目活动之间的时间依赖关系有哪四种类型？默认是哪种关系？
5. CPM中如何计算活动的最早开始日期(ES)和最晚完成日期(LF)？
6. CPM关键路径法的完整计算步骤是什么？
7. 在挣值分析中，如何判断项目成本和进度的绩效？

**学习线索**:
- WBS最低层次是"工作包"，每个工作包应有明确交付物、工作内容、责任主体、预算要求
- EVA需要三种费用值：PV（计划价值）、EV（挣值）、AC（实际成本）
- RACI：R(执行)、A(负责)、C(咨询)、I(通知)，A一般只有一个
- 时间关系：FS(完成到开始)、SS(开始到开始)、FF(完成到完成)、SF(开始到完成)，默认FS0
- 关键路径：总时差(TF)为零的活动构成的路径，决定项目工期

## Flagged ambiguities

- WBS与RAM的关联关系需要在具体案例中理解
- CPM中各种时间关系的转换（SS/FF/SF转FS）较复杂，需要重点练习
- EVA中工作完成程度的判断标准（0-100规则、50-50规则等）需注意
- 资金等值换算公式较多，需要系统记忆和应用
- 内部收益率的线性插值法求解需要理解其几何意义
- 贝叶斯公式中先验概率、后验概率、似然率的关系需要理清

## Task 1: 学习总结 ✅

### 已掌握概念

- **经济学定义**: 研究个人、企业、政府等组织如何在社会内进行选择，以及这些选择如何决定社会稀缺资源使用的科学
- **机会成本**: 资源投入某一用途后放弃的在其他用途能获得的最大收益
- **比较优势**: 取决于机会成本而非绝对成本
- **经济学四大基本问题**:
  1. 生产什么和生产多少
  2. 如何生产
  3. 为谁生产
  4. 经济体系问题（资源总量是否充分利用）
- **微观 vs 宏观**:
  - 微观: 研究单个经济单位，解决前三个问题
  - 宏观: 研究国民经济总过程，解决第四个问题
- **理性经济人假设**: 利己但不损人，追求利益最大化
- **资源配置四种方式**:
  - 自给经济: 习惯和传统
  - 市场经济: 分散决策，价格机制
  - 计划经济: 集中调节，指令性计划
  - 混合经济: 结合集中与分散决策
- **马斯洛需求层次**: 生理需求 → 安全需求 → 情感与归属感需求 → 尊重需求 → 自我实现需求
- **经济学发展史**: 古代 → 启蒙阶段（重商主义、重农主义）→ 古典经济学 → 新古典经济学 → 凯恩斯主义 → 后凯恩斯时代

## Task 2: 学前问卷

### 引导型问题

1. **什么是"消费者"和"生产者"在经济学中的定义？**
   > 学习线索：消费者具有什么基本假定？生产者的目标是什么？

2. **什么是"预算约束"？它如何影响消费者的选择？**
   > 学习线索：预算约束线上的每一点代表什么？

3. **需求曲线为什么向右下方倾斜？**
   > 学习线索：需求量的变化和需求的变化有什么区别？

4. **什么是"市场均衡"？价格如何调节市场达到均衡？**
   > 学习线索：供给和需求的相互作用如何决定均衡价格和均衡数量？

5. **政府干预（如价格管制、税收）对市场均衡有什么影响？**
   > 学习线索：价格上限和价格下限分别会导致什么市场结果？

## Task 3: 学前问卷

### 引导型问题

1. **什么是"边际报酬递减律"？它与"规模报酬"有什么区别？**
   > 学习线索：边际产量递增何时转为递减？总产量、平均产量、边际产量三者之间的关系是什么？

2. **什么是"等产量线"和"等成本线"？它们如何确定生产要素的最佳组合？**
   > 学习线索：在给定产量下如何使成本最小？在给定成本下如何使产量最大？

3. **利润最大化的必要条件是什么？边际分析在实际决策中如何应用？**
   > 学习线索："边际收益 = 边际成本"在什么情况下成立？盈亏平衡点如何计算？

4. **四种市场结构（完全竞争、垄断竞争、寡头垄断、完全垄断）的主要区别是什么？**
   > 学习线索：卖者数量、产品性质、市场进入障碍如何决定市场结构？

5. **什么是"市场失灵"？造成市场失灵的主要原因有哪些？**
   > 学习线索：外部性、信息不对称、公共产品如何导致市场机制失效？

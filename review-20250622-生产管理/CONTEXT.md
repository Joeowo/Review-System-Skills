# 共享概念上下文

## 复习范围

本次复习涵盖经济管理课程中生产管理相关的四个章节：

1. **产品开发管理** - 新产品设计、开发流程管理
2. **生产系统结构与战略** - 生产系统设计、产能规划
3. **生产与供应链管理** - 运营优化、供应链协调
4. **质量管理** - 质量控制体系、持续改进

## 学习目标

- 理解产品开发的全流程管理要点
- 掌握生产系统设计的基本原理
- 了解供应链管理的核心概念
- 熟悉质量管理的方法论和工具

## 术语表

| 术语 | 定义 |
|------|------|
| IPD | 集成产品开发，强调跨职能协同开发和并行工程 |
| CE | 并行工程，将串行作业转变为并行作业的开发模式 |
| QFD | 质量功能展开，将客户需求转化为技术要求的方法 |
| 质量屋 | QFD的基本工具，包含7个区域的广义矩阵 |
| DFM | 面向制造的产品设计，将设计要求与制造能力相匹配 |
| DFA | 面向装配的产品设计，减少装配复杂性 |
| DFE | 面向环境的设计，设计对环境友好的产品 |
| UCD | 以用户为中心的设计，产品策略以满足用户需求为动机 |
| 众包 | 产品概念来源的新型创新模式，借助社会资源提升创新实力 |
| 反向工程 | 购买竞争者产品进行研究，获得改进思路的方法 |

## Task 1: 学前问卷

### 引导型问题

**问题1：产品开发有哪些新特点？**
> 学习线索：关注用户需求变化、创新性、环境、可用性、开发速度、产品服务融合等方面
> 提示：资料列举了8个显著特点

**问题2：产品开发的思路可以从哪些渠道获得？**
> 学习线索：区分企业内部来源和外部来源
> 提示：内部如研发、市场、生产人员；外部如顾客、科研机构、供应商、竞争者

**问题3：什么是IPD（集成产品开发）？它与传统串行开发有何不同？**
> 学习线索：关注跨职能协同和并行工程
> 提示：IPD强调市场、设计、工艺、采购、制造等部门同步参与

**问题4：QFD（质量功能展开）是如何将客户需求转化为技术要求的？**
> 学习线索：质量屋的7个区域和6步构造过程
> 提示：从"whats"（用户需求）到"hows"（技术手段）的转化

**问题5：面向制造与装配的设计（DFM/DFA）为何重要？**
> 学习线索：可制造性的影响因素和模块化设计
> 提示：考虑材料选择、设计复杂度、装配便利性等因素

## Task 1: 问卷答案记录 ✅

**问题1：产品开发有哪些新特点？**
> ✅ 满足用户多方位需求、创新性、关注环境、方便使用(以顾客为中心)、减少开发时间、产品与服务结合、设计生产消费融合、产品概念来源扩展

**问题2：产品开发的思路可以从哪些渠道获得？**
> ✅ 内部：研发设计、市场营销、生产人员；外部：顾客、科研机构、供应商、竞争者

**问题3：什么是IPD与传统串行开发的区别？**
> ✅ IPD强调跨职能协同、结构化流程、全要素管理、并行工程、综合优化设计、减少纠错成本、跨职能团队直接沟通、制造和用户支持提早介入、赋予设计团队自治权

**问题4：QFD如何将客户需求转化为技术要求？**
> ✅ 质量屋A-G区：A&B用户需求、C竞争性评价、D技术手段、E关系矩阵、F技术方案相关性、G产品设计目标

**问题5：DFM/DFA为何重要及提高途径？**
> ✅ 重要性：考虑方便经济制造、降低成本保证质量缩短周期；途径：简化设计、便于装配、标准化通用化、模块化设计

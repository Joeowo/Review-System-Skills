## 已确认概念

**GDP（国内生产总值）**：
- 定义：一国在一定时期内生产的所有最终产品和劳务的市场价值
- 四个核算要点：最终产品、生产时间、流量非存量、市场价值
- 三种计算方法：生产法、收入法、支出法
- 支出法公式：GDP = C + I + G + NX

**名义GDP vs 真实GDP**：
- 名义：用当期价格计算
- 真实：用基期价格计算
- GDP平减指数 = 名义GDP / 真实GDP × 100%

**CPI（消费物价指数）**：
- 公式：一篮子商品当期价值 / 一篮子商品基期价值 × 100%
- 我国CPI包含8大类、700个代表品种
- 房价不计入CPI（属投资品）

**失业分类**：
- 按意愿：自愿失业、非自愿失业
- 按原因：周期性失业、摩擦性失业、结构性失业
- 自然失业 = 摩擦性失业 + 结构性失业

**通货膨胀**：
- 按程度：温和型（<10%）、奔腾型（10%-100%）、超级型（>100%）
- 按影响：平衡的、不平衡的
- 按预期：预期到的、未预期到的
- 费雪方程：π = m - y + v，通胀主要由货币增长率决定

**经济周期**：
- 内生理论：原因在经济体系内部
- 外生理论：原因在经济体系外部
- 奥肯定律：GDP每增长2%，约降低1%失业率（但今天已无法准确预测）

## 管理思想演变核心概念

**科学管理（泰勒）**：
- 七大原则：工作定额、挑选培养工人、标准化、差别计件工资、计划与执行分开、职能工长制、例外原则
- 核心思想：用科学方法取代传统经验，提高劳动生产率

**管理过程理论（法约尔）**：
- 五项管理职能：计划、组织、指挥、协调、控制
- 六种基本活动：技术、商业、财务、安全、会计、管理
- 14条管理原则：分工、责权一致、等级明确等
- 管理具有普遍性，可通过教育获得

**官僚制度（韦伯）**：
- 三类合法权力：法定权力（依法发布命令）、传统权力（世袭惯例）、超凡权力（个人魅力）
- 只有法定权力能作为行政组织基础：提供连续性、合理合法、清晰明确
- 权力制度化、非人格化

**霍桑实验**：
- 四阶段：照明实验、福利实验、访谈调查、群体实验
- 核心发现：工人是"社会人"而非"经济人"；存在"非正式群体"影响行为
- 霍桑效应：受试者因意识到被观察而改变行为

**X-Y理论（麦格雷戈）**：
- X理论：人性本恶，需要控制；适合物质匮乏时代
- Y理论：人性向善，可自我实现；适合物质丰裕时代
- 从外部激励转向内在激励

**管理理论四条主线**：
- ①科学管理→管理科学学派（定量分析、系统方法）
- ②人文主义→行为科学→组织行为、人力资源管理
- ③管理过程理论→一般管理理论（职能、角色、战略）
- ④官僚制度→组织理论（组织与环境、跨文化管理）

## 管理学核心概念

**管理**：
- 定义：管理者在动态环境下通过对组织内部资源及整合的组织外部资源进行有效计划、组织、控制和领导，以实现组织目标的科学和艺术的过程
- 五个内涵：服务组织目标、科学+艺术、相关联职能、内外部资源、动态环境

**管理四项职能**：
- 计划：预测未来并制订行动方案（首要职能）
- 组织：建立合理有效的组织架构
- 领导：对被管理者施加影响以实现目标
- 控制：保证组织活动符合预先制订的计划
- 关系：相互联系、相互制约、循环往复

**管理环境**：
- 外部环境：任务环境（微观）+ 一般环境（宏观）
- 内部环境：资源环境、物理环境、心理环境、文化环境

**波特五力模型**：
- 进入威胁、替代品威胁、供应商议价能力、购买者议价能力、竞争厂商竞争

**宏观环境七因素（PEST+）**：
- 政治及法律环境、宏观经济环境、社会文化环境、社会生产力、人口水平、全球化环境、自然环境

**管理者层次**：
- 基层管理者：直接指挥和监督现场作业人员
- 中层管理者：承上启下，制定具体计划
- 高层管理者：对组织负全责，决定大政方针

**明茨伯格10种管理者角色**：
- 人际角色：代表人、领导者、联络员
- 信息角色：监听者、传播者、发言人
- 决策角色：企业家、局面控制者、资源配置者、谈判者

**管理技能**：
- 概念技能：对复杂情况的洞察、分析、判断、抽象和概念化能力（高层最重要）
- 人际技能：交流、说服、管理冲突、激励、指导、谈判和领导（各层级同等重要）
- 技术技能：掌握与特定岗位有关的专业知识和技能（基层最重要）

**卢桑斯管理活动四分类**：
- 传统管理：计划、决策和控制
- 沟通：交流例行信息和处理案头工作
- 人力资源管理：激励、奖惩、处理冲突、人员配备和培训
- 网络联系：社交活动、政治活动以及与外界的交往

**有效管理者 vs 成功管理者**：
- 有效管理者：44%时间用于沟通，26%用于人力资源（打造高效团队）
- 成功管理者：48%时间用于网络联系（快速晋升的关键）

## Task 7: 学前问卷

✅ **已完成**（2026-06-21，1轮，13个问题）

**掌握要点**:
- 计划内容"5W2H1E"：What（目的内容）/Who（相关人员）/Where（场所）/When（时间范围）/Why（理由）/How（方法方案）/How much（预算）/Effect（预期效果）
- 计划四大作用：生存与发展纲领（预测变化）、组织协调前提（指明方向）、指挥实施准则（减少浪费）、控制依据（设立标准）
- 计划七大特点：主导性（指导其他职能）、普遍性（各层级都需）、连续性（周而复始）、前瞻性（面向未来）、经济性（追求效益）、创新性（与时俱进）、和谐性（动态平衡）
- 计划分类：按广度（战略计划/作业计划）、按时间（长期5年+/中期1-5年/短期<1年）、按明确程度（具体性/指导性）、按使用频率（一次性/持续性）
- 计划层次体系8个（从抽象到具体）：使命/目的 → 目标 → 战略 → 政策 → 程序 → 规则 → 方案 → 预算
- 计划编制8步骤：估量机会 → 确定目标 → 确定前提条件 → 拟定备选方案 → 评价备选方案 → 选择可行方案 → 拟定辅助计划 → 编制预算
- 目标管理（MBO）：以目标为导向、以人为中心、以成果为标准，上下级共同制定目标，明确职责，评价成果及奖励
- SMART原则：明确性、可衡量性、可实现性、相关性、时限性
- 目标管理优点：克服集权弊端（自我控制）、提高成效（客观评价）、提高员工能力、改善人际关系
- 目标管理局限性：不易量化难定目标、增加管理成本、奖惩公正性难保证、外部变化致目标短期化、目标修正不灵活
- 目标管理4程序：建立完整目标体系 → 明确责任 → 组织实施 → 检查和评价

**掌握要点**:
- 组织职能六个内容：职位设计、部门划分、职权配置、人力资源管理、协调整合、组织变革
- 组织结构设计六关键因素：工作专业化、部门化、命令链、管理跨度、集权与分权、正规化

- 命令链与统一指挥：命令链是不间断职权路线；统一指挥是一个员工对应一个主管

- 组织结构六种类型：简单结构、官僚结构、职能结构、事业部结构、矩阵结构、新型组织

- 人力资源管理流程：制定计划→明确现状→需求分析→招募选拔→培训开发→管理考评→调整流动
- 绩效评估：目的（考核、激励、改进）；环节（确定指标→选择方式→定期评估→奖励改进淘汰）

## Task 9: 学前问卷

✅ **已完成**（2026-06-21，2轮，共10个问题）

**掌握要点**:
- 领导与管理本质区别：管理基于制度结构（按规章制度、层次职位关系），领导基于影响力（打破实体角色、激发热情信心）
- 领导定义：通过对人们的有效影响而使其具有强烈意愿并努力达成组织目标的艺术或过程
- 领导实质：影响——人与人之间的互动关系
- 沟通9种障碍：无计划沟通、表达不清或理解不清、文化或语言障碍、偏见、个人情绪、过滤、不良沟通方式、信息过量、其他（性别年龄专业差异）
- 沟通7种克服方法：充分准备、理清思路善于倾听、用双方理解语言、去除偏见、设置监督机构、选择合适方式、适量信息
- 沟通本质：信息交换并被理解的过程，需要双向和反馈形成循环
- 双因素理论：保健因素（监督质量、管理方式、报酬、环境、人际关系、安全）防止不满；激励因素（赏识、升职、认同、成就、责任感）产生满意
- 麦克利兰需求理论：成就需求（接受挑战、现实态度、敢于担当）、权力需求（渴望领导地位、健谈冷静）、归属需求（合群合作、厌恶竞争、乐于助人）
- 公平理论：个人报酬/个人投入 = 他人报酬/他人投入，关注公平性对积极性的影响
- 期望理论：激励 = 效价 × 期望，动力由成果价值和达成信心共同决定
- 绩效公式：P = M × A，激励是必备条件但非全部，能力同样重要
- 承诺是激励的延伸，让员工自愿全心全意投入工作

- 企业发展三阶段：工场手工业时期（16-17世纪资本主义萌芽）、工厂生产时期（18世纪工业革命大机器生产）、企业生产时期（19世纪末-20世纪初科学管理方法出现）
- 独资企业：非法人型（无限责任、个人资产偿债）、法人型（国有独资、外商独资、一人有限责任公司）；优点（开办费用低、灵活、不交企业所得税）；缺点（无限责任、筹资有限、存续受限）
- 合伙企业：普通合伙（无限连带责任）、有限合伙（普通合伙人无限、有限合伙人有限责任）；特殊普通合伙（有过错者无限、无过错者仅出资份额为限）；优点（资本来源广、信用增强）；缺点（决策延误、存续不稳定、规模有限）
- 公司制企业：股份有限公司（股东2人以上无上限、股份自由转让、筹资强）、有限责任公司（股东不超过50人、股权严格限制）；优点（所有权与管理权分离、无限存续、有限责任、筹资强）
- 现代企业制度四特征：产权清晰（投资者是所有者、资产运营权归企业法人）、权责明确（以出资额为限承担责任和享受权益）、政企分开（企业按市场规则独立运行）、管理科学（建立科学管理制度）
- 公司法人治理结构：股东大会（最高权力机构-决策权）、董事会（执行权）、监事会（监督权）、执行机构/CEO（执行权）
- 产权制度核心：出资者所有权与法人财产权分离；出资者享有投资收益权和重大决策权，企业法人拥有独立的法人财产权
- 个体工商户：自然人形式非法人型经营单位、就其全部财产承担无限责任、不归类为企业；5000余万户，市场活动重要力量

- 企业制度三基本内容：产权制度（界定财产权、是基础）、组织制度（组织形式和权责分配）、管理制度
- 小型微型企业：占75%以上、同样资金投入吸纳就业比大中型企业多4-5倍、是解决劳动力就业的主力
- 家族企业：中国私营企业绝大多数、总产值达GDP一半以上、更抗拒其他资本介入、治理传承去家族化转型是社会热点
- CEO vs 总裁：CEO更多强调职能与责任、总裁更多强调身份地位；都由董事长宣布任命
- 各类企业占比：美国独资企业占75%、中国独资企业占80%以上（算上个体工商户个人独资市场主体占90%以上）；美国合伙企业不超过7%、中国合伙企业占比不超过2%

**掌握要点**:
- 人力资源管理九大核心职能：人力资源计划、工作分析、胜任力评价、招聘选拔、绩效管理、薪酬管理、人力资源开发、职业生涯管理、员工关系管理
- 人力资源管理三阶段：人事管理（服务与控制）→ 人力资源管理（专业化）→ 战略性人力资源管理（战略伙伴）
- 财务管理三大目标：利润最大化、股东财富最大化、企业价值最大化（最优，平衡风险与报酬、兼顾各方利益、考虑长期发展和社会责任）
- 财务管理五大原则：资金合理配置、收支平衡、收益风险均衡、成本效益、利益关系协调
- 财务管理五个基本环节：财务预测、财务决策、财务计划、财务控制、财务分析
- 物资管理两种成本：订购成本（与订购次数有关）、库存持有成本（与库存量有关）；经济订货批量使库存总成本最小
- 企业信息系统四类：TPS（基层业务处理）→ MIS（结构化业务管理）→ DSS（半结构化决策支持）→ ESS（高层战略决策）
- 数据两类：结构化数据（可用数据或统一结构表示）、非结构化数据（难以用关系数据库二维逻辑表表示）
- 数据资产三特征：能被拥有和控制、可用货币计量、能为核心业务带来经济利益
- 生产与运作管理：产品开发六阶段（规划、概念开发、系统级设计、详细设计、测试改进、生产爬坡）
- 车间布置三种方式：工艺专业化、产品专业化、固定式布置
- 供应链管理循环：计划—采购—制造—配送—回收—新一轮计划
- 质量管理四阶段：质量检验阶段→统计质量控制阶段→全面质量管理阶段→标准化质量管理阶段
- 全面质量管理（TQM）：全员、全过程、全面的质量保证体系
- 薪酬管理两类报酬：经济性报酬（工资、奖金、福利）、非经济性报酬（学习成长机会、认可、地位）
- 研发六种类型：基础研究、应用开发、项目开发、产品开发、解决方案和系统集成、服务和运营
- R&D区别：R侧重基础原理和技术路径，D侧重技术成果的产业化
- 营销观念演进：生产观念→产品观念→推销观念→营销观念→社会营销观念
- 营销组合：4P（产品、价格、地点、促销）→ 4C（顾客需求、顾客成本、便利、沟通）→ 4R（关系、节省、关联、回报）
- 市场营销过程五步骤：分析市场机会、选择目标市场、制定营销战略、设计营销组合、实施与营销策划活动
- 企业管理定义：运用计划、组织、指挥、协调、控制等手段，有效利用人、财、物等资源，通过为市场提供产品或服务，发挥最大经营效率，实现企业预期目标

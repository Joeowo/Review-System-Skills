### Task 2: GDP、失业、通胀与经济周期

**状态**: completed
**来源**: sources/04-宏观经济分析.md (第206-727行)
**学习内容**:
- GDP的概念、构成和计算方法
- 失业类型和充分就业
- 通货膨胀的衡量（CPI）和影响
- 经济周期的四个阶段

**学前问卷**:
1. GDP的三种计算方法是什么？
2. CPI是如何计算的？
3. 经济周期包含哪四个阶段？

**Q&A 轮次**:
- 第1轮 (10个问题) - GDP核算要点、名义GDP与真实GDP、失业分类、自然失业、通胀类型、费雪方程、通胀分类、经济周期理论、奥肯定律、凯恩斯与太阳黑子理论

**学习要点摘要**:
- GDP核算四个要点：最终产品、生产时间（非出售时间）、流量非存量、市场价值（非市场活动不计）
- 支出法公式：GDP = C + I + G + NX
- CPI = 一篮子商品当期价值/一篮子商品基期价值 × 100%
- 失业分类：按意愿（自愿/非自愿）、按原因（周期性/摩擦性/结构性）
- 自然失业 = 摩擦性失业 + 结构性失业（不影响经济正常运行）
- 通胀分类：按程度（温和/奔腾/超级）、按影响（平衡/不平衡）、按预期（预期/未预期）
- 费雪方程 π = m - y + v，通胀主要由货币增长率决定
- 经济周期理论：内生（经济自身因素）vs 外生（外部因素）

**学习要点摘要**:

- 公开市场业务是最主要手段：灵活、隐蔽、可预测

- 需求拉动型通胀对策：紧缩性财政（减支增税）+ 紧缩性货币（提率卖债）
- 政府购买计入GDP，转移支付不计入GDP
- 税收三特征：无偿性、强制性、固定性
- 四大政策目标：充分就业、价格稳定、经济增长、国际收支平衡
- 逆风向行事：不景气→扩张，过热→紧缩
- 法定准备金缺点：强制性强、影响大、预期影响大，很少使用
- 我国利率工具：基准利率、存贷款利率、利率结构

**学前问卷**:

**Q&A 轮次**:
- 第1轮 (13个问题) - 计划内容5W2H1E、计划作用、计划特点、计划分类（广度/时间/明确程度）、计划层次体系8个层次、目标管理MBO概念、SMART原则、目标管理优缺点、计划编制8步骤、传统vs现代计划方法

**学习要点摘要**:
- 计划内容"5W2H1E"：What（目的内容）/Who（相关人员）/Where（场所）/When（时间范围）/Why（理由）/How（方法方案）/How much（预算）/Effect（预期效果）
- 计划四大作用：生存与发展纲领（预测变化）、组织协调前提（指明方向）、指挥实施准则（减少浪费）、控制依据（设立标准）
- 计划七大特点：主导性（指导其他职能）、普遍性（各层级都需）、连续性（周而复始）、前瞻性（面向未来）、经济性（追求效益）、创新性（与时俱进）、和谐性（动态平衡）
- 计划分类：按广度（战略计划/作业计划）、按时间（长期5年+/中期1-5年/短期<1年）、按明确程度（具体性/指导性）、按使用频率（一次性/持续性）
- 计划层次体系8个（从抽象到具体）：使命/目的 → 目标 → 战略 → 政策 → 程序 → 规则 → 方案 → 预算
- 计划编制8步骤：估量机会 → 确定目标 → 确定前提条件 → 拟定备选方案 → 评价备选方案 → 选择可行方案 → 拟定辅助计划 → 编制预算
- 传统计划方法（稳定环境）：定额换算法、系数推倒法（比例法）、经验平衡法
- 现代计划方法（复杂多变）：运筹学法、滚动计划法、甘特图法、网络计划技术、计划-规划-预算方法
- 目标管理（MBO）：以目标为导向、以人为中心、以成果为标准，上下级共同制定目标，明确职责，评价成果及奖励
- SMART原则：明确性、可衡量性、可实现性、相关性、时限性
- 目标管理优点：克服集权弊端（自我控制）、提高成效（客观评价）、提高员工能力、改善人际关系
- 目标管理局限性：不易量化难定目标、增加管理成本、奖惩公正性难保证、外部变化致目标短期化、目标修正不灵活
- 目标管理4程序：建立完整目标体系 → 明确责任 → 组织实施 → 检查和评价

**Q&A 轮次**:

**学习要点摘要**:
- 组织职能六个内容：职位设计（目标分解→活动分类→形成职位）、部门划分（活动有机组合→形成部门）、职权配置（设定管理者、规定工作范围）、人力资源管理（选拔合适人选、发挥最大潜力）、协调整合（优化组织结构→有机整体）、组织变革（适应环境变化和目标调整）
- 组织结构设计六关键因素：工作专业化（分工提高效率）、部门化（按职能/产品/地域/流程/顾客/时间组合）、命令链（不间断职权路线）、管理跨度（与组织层次成反比）、集权与分权（决策权分布）、正规化（正式规范程度）
- 管理跨度与组织层次：成反比关系；窄跨度（监督严密但费用高层次多）、宽跨度（成本低但上级压力大易失控）；现代趋势趋向扁平化
- 部门化六大标准：职能部门化、产品部门化、地域部门化、生产流程部门化、顾客类型部门化、时间部门化
- 命令链与统一指挥：命令链是高层到低层不间断职权路线；职权是发布命令并期望被执行的权力；统一指挥是一个员工对应一个主管，避免多头领导
- 集权化与分权化：集权是决策权集中于一点（适合规模小、稳定、紧急、需规范）；分权是决策权分散（适合规模大、环境复杂、需快速响应）；过度的集权或分权都会阻碍发展
- 组织结构六种类型：简单结构（小规模、身兼多职）、官僚结构（标准化、高度集权、限制创新）、职能结构（专业化部门化、难协调）、事业部结构（按产品/地域设立、独立核算）、矩阵结构（双重结构、易多头领导）、新型组织
- 新型组织三种类型：团队结构（为实现子目标组建团队）、虚拟组织（外包非核心职能、专注核心竞争，如耐克）、无边界组织（打破横向纵向界限、取缔命令链，如通用电气）
- 矩阵结构：优点（人力资源利用充分、减少官僚主义、灵活性强、促进协作）；缺点（多头领导、指令混乱、协调成本高、对管理者要求高）
- 影响组织结构设计四因素：战略目标（最重要，结构服从目标）、环境因素（稳定→机械式、不稳定→灵活）、技术因素（不同技术需不同结构）、组织规模（大规模→高专业化高正规化、小规模→相反）
- 人力资源管理流程：制定目标与计划→明确内部现状→需求分析→补充与调整→招募与选拔→培训与开发→管理工作（薪酬绩效）→考评绩效→调整员工流动
- 绩效评估：目的（考核工作、给予压力、激励改进）；环节（确定指标体系→选择评估方式→定期给出结果→奖励优秀改进落后末位淘汰）；评估方式（自我评估、员工互评、上级评定）

- 第2轮 (5个问题) - 深化问答：加薪与双因素理论应用、期望理论公式含义、沟通的双向性、绩效公式P=M×A、综合应用分析

**学习要点摘要**:
- 领导与管理区别：管理基于制度结构（按规章制度、层次职位关系），领导基于影响力（打破实体角色、激发热情信心）
- 领导定义：通过对人们的有效影响而使其具有强烈意愿并努力达成组织目标的艺术或过程；实质是影响——人与人之间的互动
- 沟通9种障碍：无计划沟通、表达不清或理解不清、文化或语言障碍、偏见、个人情绪、过滤、不良沟通方式、信息过量、其他（性别年龄专业差异）
- 沟通7种克服方法：充分准备、理清思路善于倾听、用双方理解语言、去除偏见、设置监督机构、选择合适方式、适量信息
- 沟通本质：信息交换并被理解的过程，需要双向和反馈形成循环
- 双因素理论：保健因素（监督质量、管理方式、报酬、环境、人际关系、安全）防止不满；激励因素（赏识、升职、认同、成就、责任感）产生满意
- 麦克利兰需求理论：成就需求（接受挑战、现实态度、敢于担当）、权力需求（渴望领导地位、健谈冷静）、归属需求（合群合作、厌恶竞争、乐于助人）
- 公平理论：个人报酬/个人投入 = 他人报酬/他人投入，关注公平性对积极性的影响
- 期望理论：激励 = 效价 × 期望，动力由成果价值和达成信心共同决定
- 绩效公式：P = M × A，激励是必备条件但非全部，能力同样重要；承诺是激励的延伸

**学前问卷**:
1. 企业的四个基本特征是什么？企业发展经历了哪三个阶段？
2. 独资企业、合伙企业、公司制企业的主要区别是什么？各有什么优缺点？
3. 什么是现代企业制度？它有哪些特征？
4. 公司法人治理结构包括哪些组成部分？各自的职权是什么？

**学习线索**:
- 企业是从事商品生产、流通及服务性活动，以营利为目标的基本经营单位
- 企业法律形式的核心区别在于：法人资格、责任承担（无限/有限）、筹资能力
- 现代企业制度的核心是产权清晰——出资者所有权与法人财产权分离
- 公司法人治理结构实现"三权分立"：决策权（股东大会）、执行权（董事会和管理层）、监督权（监事会）

**Q&A 轮次**:
- 第1轮 (18个问题) - 企业四个基本特征、企业发展三阶段、三种企业法律形式的区别、现代企业制度四特征、法人治理结构、股份有限公司vs有限责任公司、产权制度核心、特殊普通合伙企业、CEOvs总裁、各类企业优缺点、个体工商户、企业按自然属性分类、企业制度三内容、小型微型企业数据、家族企业地位、各类企业占比

**学习要点摘要**:
- 企业四个基本特征：从事商品生产流通及服务性活动、满足社会需要并获取利润、实行自主经营自负盈亏自我发展自我约束、独立享受权利和承担义务的法人或非法人型基本经营单位
- 企业发展三阶段：工场手工业时期（16-17世纪资本主义萌芽）、工厂生产时期（18世纪工业革命大机器生产）、企业生产时期（19世纪末-20世纪初科学管理方法出现）
- 独资企业：非法人型（无限责任、个人资产偿债）、法人型（国有独资、外商独资、一人有限责任公司）；优点（开办费用低、灵活、利润归个人、不交企业所得税）；缺点（无限责任、筹资有限、存续受限）
- 合伙企业：普通合伙（无限连带责任）、有限合伙（普通合伙人无限、有限合伙人有限责任）；特殊普通合伙（有过错者无限、无过错者仅出资份额为限）；优点（资本来源广、信用增强）；缺点（决策延误、存续不稳定、规模有限）
- 公司制企业：股份有限公司（股东2人以上无上限、股份自由转让、筹资强）、有限责任公司（股东不超过50人、股权严格限制、筹资相对弱）；优点（所有权与管理权分离、股份自由转让、无限存续、有限责任、筹资强）；缺点（创办复杂、监管多、监督可能弱）
- 现代企业制度四特征：产权清晰（投资者是所有者、资产运营权归企业法人）、权责明确（以出资额为限承担责任和享受权益）、政企分开（企业按市场规则独立运行）、管理科学（建立科学管理制度）
- 公司法人治理结构：股东大会（最高权力机构、决策权：选举董事监事、审议预算、决定合并解散）、董事会（执行权：执行决议、制订经营计划、任免经理）、监事会（监督权：财务检查、监督董事高管）、执行机构/CEO（执行权：组织实施董事会决议、实施经营计划）
- 产权制度核心：出资者所有权与法人财产权分离；出资者享有投资收益权和重大决策权，企业法人拥有独立的法人财产权，自主经营自负盈亏
- 个体工商户：自然人形式非法人型经营单位、就其全部财产承担无限责任（包括与配偶共同拥有的私人住房）、不归类为企业；5000余万户，市场活动重要力量
- 企业按自然属性分类：生产要素集约程度（劳动/资本/技术/知识密集型）、经营内容和行业（工业/农业/商业/金融等）、生产资料所有制（国有/集体/私营/中外合资等）、生产力组织形式（单一/多元/企业集团、本土/跨国）、企业规模（大型/中小型）、家族控制（家族/非家族）
- 企业制度三基本内容：产权制度（界定财产权、是基础）、组织制度（组织形式和权责分配）、管理制度；产权制度是决定组织和管理制度的基础
- 小型微型企业：截至2015年4月企业1900余万户、小型微型企业占75%以上、同样资金投入吸纳就业比大型中型企业多4-5倍、是解决劳动力就业的主力
- 家族企业：中国私营企业绝大多数是家族企业、总产值达GDP一半以上、中国经济主要力量、更抗拒其他资本介入、治理传承去家族化转型是社会热点
- CEO vs 总裁：CEO更多强调职能与责任、总裁更多强调身份地位；有些企业既有CEO也有总裁、有些企业将两者合为"总裁兼首席执行官"；都由董事长宣布任命、董事长也可能兼任
- 各类企业占比：美国独资企业占75%、中国独资企业占80%以上（算上个体工商户个人独资市场主体占90%以上）；美国合伙企业不超过7%、中国合伙企业占比不超过2%

**学习要点摘要**:

- 人力资源管理三阶段：人事管理（服务与控制）→ 人力资源管理（专业化）→ 战略性人力资源管理（战略伙伴）
- 财务管理三大目标：利润最大化、股东财富最大化、企业价值最大化（最优，平衡风险与报酬、兼顾各方利益、考虑长期发展和社会责任）
- 财务管理五大原则：资金合理配置、收支平衡、收益风险均衡、成本效益、利益关系协调
- 财务管理五个基本环节：财务预测、财务决策、财务计划、财务控制、财务分析
- 物资管理两种成本：订购成本（与订购次数有关）、库存持有成本（与库存量有关）；经济订货批量使库存总成本最小
- 企业信息系统四类：TPS（基层业务处理）→ MIS（结构化业务管理）→ DSS（半结构化决策支持）→ ESS（高层战略决策）
- 数据两类：结构化数据（可用数据或统一结构表示）、非结构化数据（难以用关系数据库二维逻辑表表示）
- 数据资产三特征：能被拥有和控制、可用货币计量、能为核心业务带来经济利益
- 生产与运作管理：产品开发六阶段（规划、概念开发、系统级设计、详细设计、测试改进、生产爬坡）
- 车间布置三种方式：工艺专业化、产品专业化、固定式布置
- 供应链管理循环：计划—采购—制造—配送—回收—新一轮计划
- 质量管理四阶段：质量检验阶段→统计质量控制阶段→全面质量管理阶段→标准化质量管理阶段
- 全面质量管理（TQM）：全员、全过程、全面的质量保证体系
- 薪酬管理两类报酬：经济性报酬（工资、奖金、福利）、非经济性报酬（学习成长机会、认可、地位）
- 研发六种类型：基础研究、应用开发、项目开发、产品开发、解决方案和系统集成、服务和运营
- R&D区别：R侧重基础原理和技术路径，D侧重技术成果的产业化
- 营销观念演进：生产观念→产品观念→推销观念→营销观念→社会营销观念
- 营销组合：4P（产品、价格、地点、促销）→ 4C（顾客需求、顾客成本、便利、沟通）→ 4R（关系、节省、关联、回报）
- 市场营销过程五步骤：分析市场机会、选择目标市场、制定营销战略、设计营销组合、实施与营销策划活动
- 企业管理定义：运用计划、组织、指挥、协调、控制等手段，有效利用人、财、物等资源，通过为市场提供产品或服务，发挥最大经营效率，实现企业预期目标

**评价**: 理解全面准确，表述清晰，已达标

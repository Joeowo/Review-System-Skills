**记忆要点：**

```
经济学 = 稀缺性 → 选择 → 效率
管理学 = 计划 → 组织 → 领导 → 控制
```

### 2️⃣ Chp02 微观市场机制分析（74页）

**核心概念：**
- 需求函数：QD = f(P, 收入, 相关商品价格, 偏好, 预期)
- 供给函数：QS = f(P, 生产成本, 技术, 要素价格, 预期)
- 市场均衡：供给 = 需求时的价格和数量
- 需求弹性：Ed = (ΔQ/Q) / (ΔP/P) = 价格变化1%带来的需求变化百分比
- 供给弹性：Es = (ΔQ/Q) / (ΔP/P)

**核心公式：**
```
均衡条件：QD = QS
需求价格弹性：Ed = -ΔQ/ΔP × P/Q
供给价格弹性：Es = ΔQ/ΔP × P/Q
```

**记忆要点：**
- 需求定律：价格↑，需求↓
- 供给定律：价格↑，供给↑
- 弹性越大，对价格变化越敏感

### 3️⃣ Chp03 要素投入与市场结构（80页）

**核心概念：**
- 生产函数：Q = f(K, L, T, M) - 产出与投入要素的关系
- 边际产量（MP）：增加一单位投入带来的产出增量
- 边际报酬递减律：其他要素不变时，连续增加某要素投入，边际产量先增后减
- 边际技术替代率（MRS）：MRSxy = MPx/MPy
- 等成本线：Px·x + Py·y = C
- 成本函数：TC = FC + VC
- 利润最大化条件：MR = MC（边际收益 = 边际成本）

**核心公式：**
```
边际技术替代率：MRS = |dy/dx| = MPx/MPy
等成本条件：Px·x + Py·y = C
切点条件（最优投入）：|dy/dx| = MPx/MPy = Px/Py
利润最大化：MR = MC
```

**记忆要点：**
- 边际量 = 导数概念
- 利润最大点 = MR = MC
- 完全竞争 vs 垄断的效率差异

### 4️⃣ Chp06 管理的职能（90页）

**核心概念：**
- 管理四大职能（按顺序）：
  1. **计划**（Planning）：设定目标，制定战略
  2. **组织**（Organizing）：分配资源，建立结构
  3. **领导**（Leading）：激励员工，指导工作
  4. **控制**（Controlling）：监控绩效，纠正偏差

- 一般管理理论特点：关注管理职能和管理原则

**记忆要点：**
```
计划（做什么）→ 组织（谁来做）→ 领导（让人去做）→ 控制（做得如何）
```

### 6️⃣ Chp10 库存与生产计划管理（152页）⭐计算重点

**核心概念：**
- 需求预测：移动平均法、加权移动平均法
- 供应链结构：供应商 → 制造商 → 分销商 → 零售商 → 客户
- 牛鞭效应：需求信息在供应链中被逐级放大
- EOQ模型：经济订货批量，使库存总成本最小

**核心公式：**
```
简单移动平均：Ft = (Dt-1 + Dt-2 + ... + Dt-n) / n

加权移动平均：Ft = Σ(ci × Dt-i) / n，其中 Σci = n

EOQ公式：Q* = √(2AD/h)
  A = 年需求量
  D = 每次订货成本
  h = 单位存储成本
```

**记忆要点：**
- EOQ平衡：订货成本 vs 存储成本
- 牛鞭效应根源：信息不对称、批量订货、价格波动
- 安全库存：应对需求不确定性

### 7️⃣ Chp13 投资项目经济评价（92页）⭐计算重点

**核心概念：**
- 资金时间价值：今天的一元钱 > 明天的一元钱
- 静态投资回收期：不考虑时间价值
- 动态投资回收期：考虑时间价值
- 净现值（NPV）：未来现金流折现到现在的总和
- 内部收益率（IRR）：使NPV=0的折现率

**核心公式：**
```
复利终值：F = P(1 + i)^n
复利现值：P = F / (1 + i)^n

年金终值：F = A × [(1+i)^n - 1] / i
年金现值：P = A × [1 - (1+i)^(-n)] / i

净现值：NPV = Σ[Ct / (1+i)^t] - I0
  Ct = 第t期净现金流
  I0 = 初始投资
  i = 折现率

内部收益率：求 i* 使 NPV = 0
```

**判断准则：**
```
NPV ≥ 0 → 项目可行
NPV < 0 → 项目不可行

IRR ≥ 基准收益率 → 项目可行
IRR < 基准收益率 → 项目不可行
```

**记忆要点：**
- NPV是绝对值，IRR是相对值
- NPV和IRR结论可能矛盾时，以NPV为准

### 8️⃣ Chp14 决策分析（92页）⭐计算重点

**核心概念：**
- 决策三要素：方案、自然状态、收益
- 不确定型决策：不知道各状态发生的概率
- 风险型决策：知道各状态发生的概率
- 决策树：用树状图表示决策过程
- 期望值准则：选择期望收益最大的方案
- 贝叶斯决策：用新信息修正概率（先验概率 → 后验概率）

**决策树结构：**
```
     □ 决策节点
     ├── 方案A
     │   ├── 状态1 (概率p1) → 收益a
     │   └── 状态2 (概率p2) → 收益b
     └── 方案B
         ├── 状态1 (概率p1) → 收益c
         └── 状态2 (概率p2) → 收益d
```

**核心公式：**
```
期望值：E = Σ(概率 × 收益)

贝叶斯公式：P(A|B) = P(B|A) × P(A) / P(B)
后验概率 = 似然度 × 先验概率 / 证据
```

**决策准则：**
```
乐观准则（大中取大）：max(max收益)
悲观准则（小中取大）：max(min收益)
后悔值准则：min(max后悔值)
期望值准则：max(期望收益)
```

**记忆要点：**
- 决策树从右向左计算（倒推法）
- 贝叶斯 = 更新认知

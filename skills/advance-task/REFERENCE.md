## 完成标准

一个 task 标记为 `completed`，需满足：

1. ✅ 至少 2 轮 Q&A
2. ✅ 能回答所有学前问卷问题
3. ✅ 无明显待解决问题
4. ✅ 用户表示理解或通过验收问题

## ADR 模板

创建新 ADR 时，使用以下模板：

```markdown
# ADR XXXX: [决策标题]

**状态**: [提议/已采纳/已废弃/已替代]
**日期**: YYYY-MM-DD
**决策者**: [姓名/团队]

---

## 上下文

[描述触发此决策的背景和问题]

---

## 决策

[清晰陈述决策内容]

---

## 理由

[支持此决策的论据，包括替代方案和未采纳的原因]

---

## 后果

[应用此决策后的影响，包括正面、负面影响及缓解措施]

---

## 实施细节

[技术实施的具体说明]

---

## 相关决策

- [ADR XXXX: 相关决策](xxxx.md)

---

**文档版本**: 1.0
```

---

## 状态定义

| 状态 | 说明 |
|------|------|
| **提议** | 决策正在讨论中 |
| **已采纳** | 决策已批准并正在实施 |
| **已废弃** | 决策不再适用 |
| **已替代** | 被新的 ADR 替代 |
